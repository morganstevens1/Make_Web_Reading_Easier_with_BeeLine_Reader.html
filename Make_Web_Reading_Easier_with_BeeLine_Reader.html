Make Web Reading Easier with BeeLine Reader

BeeLine reader is a program that uses an "eye-guiding color gradient - to pull eyes through long blocks of text." One is able to download this as an extension onto their browser to use on different websites, on their phone, or many other programs like e-readers. BeeLine Reader is designed to support reading long blocks of text and has been proven to:

  1. Reduce eye-fatigue
  2. Aid with distance learning
  3. Faster reading speeds
Click here to view the website for more information along with a reading sample.

BeeLine Reader is available to all Bryn Mawr students, faculty, and staff:

  1. Install the plugin for Chrome or Firefox
  2. Click this registration link to register your browser.
  3. To read PDFs using BeeLine Reader, install the PDF Chrome extension.
